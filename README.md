# 🎬 Netflix Data Analysis Dashboard (Power BI Project)

## 📌 Project Overview

This project is a Netflix Data Analysis Dashboard** created using Power BI.
The main purpose of this project is to explore and understand Netflix content using data visualization.

It helps to answer important questions like:

* What type of content is available on Netflix?
* Which genres are most popular?
* How has Netflix content grown over the years?
* Which countries contribute the most content?
* What ratings are most common?

---

## 🎯 Objective of the Project

The main objective of this project is:

* To analyze Netflix movies and TV shows data
* To identify trends and patterns in content
* To improve data visualization skills using Power BI
* To convert raw data into meaningful insights

---

## 📊 Dashboard Features

This dashboard includes the following visualizations:

### 1. Summary Cards

* Total Titles
* Total Ratings
* Total Genres
* Total Countries
* Latest Release Year
* Total Cast

### 2. Movies vs TV Shows Analysis

* Shows the distribution between Movies and TV Shows
* Helps understand content type preference on Netflix

### 3. Genre Analysis

* Displays top genres based on number of titles
* Identifies most common content categories

### 4. Ratings Analysis

* Shows different content ratings like TV-MA, TV-14, PG, etc.
* Helps understand target audience type

### 5. Country Analysis

* Displays top countries producing Netflix content
* Shows global contribution to Netflix library

### 6. Yearly Trend Analysis

* Shows how Netflix content has increased over time
* Helps understand growth pattern of Netflix

---

## 🔍 Key Insights from the Data

* Movies make up the majority of Netflix content compared to TV Shows
* Netflix content increased rapidly after 2015 and peaked around 2019–2020
* TV-MA is the most common rating, showing focus on mature audience
* Drama and Documentary are the most popular genres
* United States is the leading content contributor
* Kids and Family content is less compared to other categories
